QA Testing Portfolio – SauceDemo (E-commerce Testing Project)

This project simulates real-world QA testing for an e-commerce web application.
This project demonstrates my QA testing skills using the SauceDemo website.

Scope
Modules tested:
- Login
- Product Listing
- Cart
- Checkout

Testing Types
- Manual Testing
- API Testing (Postman)

Test Artifacts (Manual Testing)
1. Test Plan – Defines testing scope and strategy  
2. Test Scenario – High-level test conditions  
3. Test Case – Detailed steps and expected results  
4. Bug Report – Issues identified during testing  
5. Defect Log – Bug tracking and status updates  
6. Test Summary Report – Final testing results  

Tools Used
- Jira (Bug Tracking)
- Postman (API Testing)
- Google Sheets / Excel

Repository Structure
- manual-testing/
- api-testing/
- jira/
